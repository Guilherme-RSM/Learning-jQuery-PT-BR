O que é o jQuery?

jQuery é uma biblioteca JavaScript rápida, pequena e rica em recursos. Isso torna coisas como travessia e manipulação de documentos HTML, manipulação de eventos e animação muito mais simples. Todo o poder do jQuery é acessado via JavaScript, portanto, ter uma forte compreensão do JavaScript é essencial para entender, estruturar e depurar seu código.

O Slogan do jQuery é Write less, do more. Que significa: Escreva menos, faça mais.
Outra grande vantagem do jQuery é que você não precisa se preocupar com o suporte do navegador, pois seu código será executado exatamente da mesma forma em todos os principais navegadores.

Primeiro, vamos dar uma olhada em um exemplo de manipulação de HTML com JavaScript . Para obter o elemento com o id="start" e alterar seu html para "Go", precisaremos fazer o seguinte:

var el = document.getElementById("start"); el.innerHTML = "Go";

Para fazer a mesma manipulação com jQuery, precisamos apenas de uma única linha de código:

$("#start").html("Go");

Posteriormente você aprenderá sobre a nova sintaxe, mas como você pode ver, o código é muito mais curto e fácil de entender.

Começando

Você pode baixar uma cópia da biblioteca jQuery em www.jquery.com ou, como alternativa, pode incluí-la em uma CDN (Content Delivery Network), como Google ou Microsoft. Usaremos o CDN do site oficial do jQuery. Para começar a usar o jQuery, primeiro precisamos adicioná-lo ao cabeçalho do nosso documento HTML usando a tag script:


          <!DOCTYPE html>
<html>
    <head>
        <title>Page Title
        <script src="https://code.jquery.com/jquery-3.1.1.js"></script>
    </head>
    <body>
    </body>
</html>

OBS: Como jQuery é uma biblioteca JavaScript, possui a extensão de arquivo .js .

É uma boa prática esperar que o documento HTML esteja totalmente carregado e pronto antes de trabalhar com ele. Para isso usamos o evento ready do objeto document:

$(document).ready(function() { // código jQuery vai aqui });

O $ é usado para acessar jQuery. A partir daqui, o código acessa o objeto documento e define uma função a ser chamada quando o evento pronto do documento for acionado. Isso evita que qualquer código jQuery seja executado antes que o documento termine de carregar. Como o código acima é usado em quase todos os casos ao usar jQuery, há um atalho útil para escrevê-lo:

$(function() { // código jQuery vai aqui });

Este código executa a mesma tarefa que o código mais longo acima.

Agora, tendo a biblioteca jQuery em nossa seção head e tendo definido o evento document ready, podemos iniciar nossa primeira manipulação jQuery! Vamos alterar o conteúdo do elemento div.

Código HTML:

<!DOCTYPE html>
 
<html>
 
  <head>
 
    <title>Título da página</title>
 
   <script src="https://code.jquery.com/jquery-3.1.1.js"></script>
          
  </head>
 
  <body>
 
    <div id="start">Start</div>
 
  </body>
 
</html>

Sintaxe

jQuery é usado para selecionar (consultar) elementos HTML e realizar "ações" neles. A sintaxe básica é:

  • O $ acessa jQuery.
  • O (seletor) encontra elementos HTML.
  • A ação() é então executada no(s) elemento(s).

$("selector").action()

Por exemplo:

$("p").hide() //oculta todos os elementos <p>
$(".demo").hide() //oculta todos os elementos com class="demo"
$("#demo").hide() //oculta o elemento com id="demo"

Olhando para o código do nosso exemplo anterior:

$("#start").html("Go!");

Isso seleciona o elemento com o id="start" e chama o método html() para ele. O método html() é usado para alterar o conteúdo HTML de um elemento.

*OBS: Se você já usou CSS antes, notará que o jQuery usa a sintaxe CSS para selecionar elementos.
